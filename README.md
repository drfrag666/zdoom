# ZZDoom
 ZZDoom is a fork of GZDoom 2.4.0 (https://github.com/coelckers/gzdoom/tree/maint2.4) without the GL renderer.
 GZDoom was itself a fork of ZDoom (https://github.com/rheit/zdoom) back then.
